# Odisha Waste  Management System

A smart, role-based waste and wastewater management platform designed to support rural and urban sanitation governance in Odisha through digital monitoring, analytics, and operational coordination.

## Overview
This project provides an integrated dashboard-driven system for multiple stakeholders including:

- Administrators  
- Government Officials  
- GP/ULB Authorities  
- Driver/ Transport

The platform helps monitor waste collection, sanitation performance, route planning, reporting, alerts, and data-driven decision-making.

## Key Features
- Real-time waste and wastewater monitoring dashboards  
- Role-based login and management system  
- GP/ULB performance tracking  
- District and block-level analytics  
- Route planning and vehicle monitoring  
- Personnel and service request management  
- Waste collection and receipt records  
- Alerts, discrepancy tracking, and reporting  
- Interactive charts, maps, and data visualization  
- Citizen interface for service-related interactions  

## Tech Stack
- Next.js  
- React  
- TypeScript  
- Firebase  
- Tailwind CSS  
- Recharts  
- Cloud Deployment Ready (GitHub + Cloudflare)

## Objective
To build a scalable digital ecosystem that improves sanitation transparency, monitoring efficiency, and sustainable waste management operations.
